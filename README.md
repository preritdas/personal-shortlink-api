# Personal Shortlink API

Personal API for the creation and distribution of shortened links. This will live under the subdomain link.preritdas.com. The root (https://link.preritdas.com) will redirect straight to preritdas.com, but custom codes after (ex. link.preritdas.com/code-here) will redirect a user.

Shortlink redirections are registered using the `/manage` router, which will have various CRUD sub-endpoints.
